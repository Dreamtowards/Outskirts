



# Syntax Sugars

## Local Variable Type Inference

Use of "auto" declarate-type-keyword. only allowed use in local scopes vardefs.
```
    // local scope:
    auto var1 = ...;
```

> why not use keyword "let", "var", "val"  
> val: loosed. wasted.  var: script dynamic language.  let: too high-level.
